# EP652 股癌節目簡報與深度分析

本報告針對《股癌 Gooaye》EP652 進行深度摘要與分析，內容涵蓋市場趨勢、投資邏輯、個股動態及生活智慧，旨在提供投資者與讀者最精確且具行動力的資訊參考。

## 執行摘要
本集核心環繞在美股與台股的族群輪動觀察。主委指出，在經歷了震盪的年度第一季後，資金已開始從單一的光通訊（Optical Communication）擴散至 CPU（伺服器與客製化 ASIC）及被動元件等族群。特別強調了 AI 時代下「算力供應失衡」不僅限於 GPU，CPU 的短缺也正成為新常態。此外，針對市場上的 AI 模型行銷語言（如 Anthropic 的 Mythos 延遲發布）提出了批判性見解，認為本質問題在於算力不足而非安全疑慮。

## 核心主題深度分析

### 1. AI 算力新戰場：從 GPU 轉向 CPU 與 ASIC
*   **趨勢觀察：** 在 Agentic AI（代理式 AI）時代，除了 GPU 提供算力，CPU 扮演著「教練」般的調度角色，需求大幅提升。
*   **關鍵數據點：** AWS 表示其客製化 CPU (Graviton) 算力已被客戶要求全包，顯示供應端極度吃緊。這將迫使客戶轉向通用型 CPU（如 Intel、AMD）尋求支援。
*   **RISC-V 的崛起：** NVIDIA 入股 SiFive，顯示市場開始關注能提供特化版 CPU 架構的 RISC-V 領域。 [逐字稿 00:12:00]

### 2. 被動元件的漲價與缺貨潮
*   **當前位階：** 處於類似 2023 或 2024 年記憶體發作前夕的階段。
*   **族群細分：**
    *   **TCAP（鉭電容）與 SPCAP（聚合物鋁電容）：** 嚴重缺貨。
    *   **MLCC（積層陶瓷電容器）：** 高階應用（如 AI 伺服器、手機、電動車）緊繃，由村田（Murata）、三星電機（Samco）主導。 [逐字稿 00:17:00]
*   **催化劑：** 預期國巨（2327）法說會將釋出正面訊號，且旗下 Kemet（基美）已持續針對特定品項調漲。 [逐字稿 00:18:00]

### 3. 光通訊供應鏈的「隱形貴人」：SMT 與 LED 廠的轉型
*   **新模式：** 因光通訊需求過大，美系巨頭直接自帶設備（假方出錢），租借現有的 SMT（表面黏著技術）或 LED 打件廠產線進行生產。
*   **投資警示：** 此類標的屬於「生水區」，雖具短期獲利爆發力，但需注意驗證階段風險與客戶隨時抽走訂單的可能性。 [逐字稿 00:19:00]
