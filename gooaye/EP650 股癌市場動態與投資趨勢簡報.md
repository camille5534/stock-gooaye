# EP650 股癌市場動態與投資趨勢簡報

## 執行摘要

本報告針對《股癌》EP650 集數內容進行深度分析與彙整。當前市場處於波動劇烈且交易難度高的狀態，美股難度尤甚於台股。核心投資焦點正從 AI 的「訓練時代」（GPU 為主）轉向「代理式 AI」（Agentic AI）的「推論時代」，這將顯著提升對 **CPU** 的需求與重視。地緣政治方面，中東局勢持續惡化，不僅影響油價，更推升了鋁價並對電子零組件供應鏈（如鋁電容）造成潛在斷鏈風險。

---

## 核心主題分析

### 1. AI 產業新典範：從 GPU 到 CPU 的價值重估

隨着 AI 應用進入 Agentic AI 時代，工作流變得更加複雜，涉及大量的推理、決策、代碼執行與 API 調用。

*   **功能轉換：** GPU 負責「砍分」（大量計算），而 CPU 則扮演「教練」（指揮、調度與推理）。
*   **需求激增：** 在傳統 AI 模型中，CPU 的負擔約僅 10%，但在 Agentic AI 中，CPU 的運作佔比可提升至 40% 到 50%，甚至在某些工作流中佔總延遲的 50% 至 90%。
*   **關鍵企業動態：**

| 企業 | 市場觀點與動態 |
| :--- | :--- |
| **NVIDIA** | 推出的 Blackwell 平台（如 NVL72）整合了強大的 CPU 解決方案，顯示其已轉型為 CPU 相關受惠股。 |
| **AMD** | 在 X86 市場快速蠶食 Intel 份額，且在 AI 訓練架構下具備高度成長性。 |
| **Intel** | 承認錯估 CPU 需求，目前產能極度受限（Absolutely constrained），顯示市場需求超預期。 |
| **ARM** | 伺服器市佔率從 2018 年的 <5% 預計在 2025 年成長至 20-25%，雲端服務供應商（CSP）如 AWS、Google、Azure 紛紛採用其架構自研晶片。 |

### 2. 地緣政治與原物料連鎖效應

中東戰火（伊朗與美國/盟友衝突）對市場產生了情緒性與實質性的雙重打擊。

*   **五角大廈披薩追蹤器（Pizza Tracker）：** 透過觀察五角大廈附近披薩店的流量，預判美國可能有重大軍事行動。
*   **鋁價與供應鏈：** 中東佔全球鋁產量約 10% 以上，隨着當地工廠受損，鋁價創下新高。這將傳導至**鋁箔**、**鋁電容器**等電子被動元件。
*   **塑化產業連鎖反應：** 伊朗廉價石油供應若被阻斷，原先依賴便宜能源的中國塑化企業競爭力將下滑，進而對台灣塑化股產生利多。

### 3. 投資策略與心態

*   **處理勝於預判：** 投資者不應追求成為「預言家」（如預測戰爭是否發生），而應專注於事件發生後的處理能力。
*   **買入理由的純化：** 買入股票前必須釐清是基於「題材」、「價值」還是「動能」。若買入理由（如 AI 題材）消失，則應果斷處置，而非在跌破 60% 後才思考停損。
*   **做空的風險：** 股市長期期望值利於做多，做空的潛在獲利上限僅 100%，但風險巨大且常面臨公司派與市場情緒的雙重擠壓。

*   **背景：** 描述近幾個月美股與台股的極高操作難度，提醒聽眾目前的震盪並非個案感觸。

> **「GPU 算是在場上一個非常強的球員，你叫他去砍分，他就瘋狂地給你砍分；CPU 就比較偏教練，在旁邊去做指揮的。」**
*   **背景：** 形象化地解釋在 Agentic AI 時代，CPU 如何從配角轉為主導推理與調度的關鍵角色。

> **「羨慕跟嫉妒別人是快樂之賊... 只要你開始會對別人的生活產生羨慕跟嫉妒的時候，你就破壞了自己的遊戲體驗。」**

3.  **操作心法調整：**
