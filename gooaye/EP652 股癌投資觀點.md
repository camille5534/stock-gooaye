## 投資心法卡

### 1. 體力是交易的基礎設施
體力被視為大腦的電源供應器。市場波動大時，判斷力下滑往往是因為身體「沒電」，導致處理器降頻。維持良好的精神狀態是確保投資決策在「均線之上」的關鍵。[逐字稿 00:00:10]

### 2. 選股勝於努力
在趨勢向下的垃圾標的中，即便擁有再好的買賣點調節技術，也難以避免淨值掉落。資金規畫應順應趨勢，將錢投入強勢族群（如近期由光通訊轉向 CPU）。[逐字稿 00:11:00]

### 3. 多頭市場的特徵：輪動與不畏高
*   **多頭行情定義：** 指多個族群輪流向上衝刺。健康的行情伴隨著類股輪動，而非單一標的獨撐。[逐字稿 00:15:30]
*   **看待股價高點：** 投資者不應怕高，而應關注「EPS 還有多少潛力」以及「上方還有多少 Rerate（重新評價，指市場給予更高本益比的過程）空間」。任何標的的歷史新高，往往都是從突破前高開始的。[逐字稿 00:30:30]

## 市場與族群趨勢卡

### 1. 市場位階分析
*   **台股：** 目前處於極強的多頭格局，指數拉至新高，表現冠居全球。[逐字稿 00:10:00]
*   **美股：** 來到壓力位階，若能站穩季線則視為重回多頭。資金已開始從光通訊擴散至 CPU 及高速傳輸族群。[逐字稿 00:10:30]

### 2. 算力需求轉移：從 GPU 到 CPU
AI 算力需求呈現「無底洞」狀態。目前市場重心開始轉向 **CPU**，因在 Agentic AI（代理式 AI）時代，CPU 需擔任「教練」角色負責調度。AWS 的 Graviton CPU 傳出供不應求。此外，RISC-V 架構也因能製作特化版 CPU 而重獲關注。[逐字稿 00:12:30]

### 3. 被動元件缺貨潮再起
被動元件（尤其是電容）出現產能吃緊，邏輯類比於 2017 年的盛況。
*   **缺貨品項：** T-Cap（鉭質電容）、SP-Cap（聚合物鋁電容）嚴重缺貨；MLCC（積層陶瓷電容）接近緊繃。[逐字稿 00:23:40]
*   **驅動因素：** 高階應用（AI 伺服器）需求排擠低階產能，類似記憶體先前的漲價邏輯。[逐字稿 00:24:20]

### 4. 光通訊產能的新趨勢：「東方貴人」模式
由於光通訊需求過大，美系巨頭直接自帶設備（假方出錢），尋找台灣擁有閒置 SMT 產線（表面黏著技術，將電子元件焊接至電路板的製程）的廠商合作生產，甚至包含 LED 或網通等「八竿子打不著」的場地。[逐字稿 00:26:00]

## 生活趣聞卡

1.  **戒菸的「深呼吸」邏輯：** 發現菸癮的本質除了尼古丁，更多是人類在壓力下需要「一個人深呼吸」的儀式。建議想戒菸者可嘗試去戶外單純深呼吸兩分鐘，也能達到放鬆大腦的效果。[逐字稿 00:05:00]
2.  **Z 世代的酷文化：** 2000 年後的年輕一代社交不再依賴酒精，甚至在麵包店辦派對喝汽水。他們更看重準時下班與自我生活，不輕易接受老派的「學徒制」或「社交菸酒習慣」。[逐字稿 00:08:00]
3.  **兒子的「指揮官」行徑：** 其次子安哥（諾亞）雖然還不會說話，但會像「巡弋飛彈」一樣精準鎖定目標（如電腦），並不斷用手指指揮大人抱他過去，展現強烈主見。[逐字稿 00:46:00]

## QA 精選

*   **關於親子關係：** 讀者分享「快樂存折」概念。主委認同教育不應教條式，而是透過分享真實感受產生共鳴。與孩子相處的核心是時間陪伴。[逐字稿 00:28:40]
*   **關於父母相處：** 建議與傳統、容易走心的父母保持「安全距離」。不需要追求「三代同堂」的虛假和諧，應優先顧好自己的核心家庭（配偶與小孩），行有餘力再向外擴張。[逐字稿 00:33:00]
*   **關於技術線型：** 「現型好」就是指股價長在右上角。雖然看重線型，但必須配合基本面支撐才有大波段。單純看線買入（如部分少年股神）停損機率較高。[逐字稿 00:31:00]
*   **關於大額交易出貨邏輯：** 當持股比例過高（單一持股達 20-30%）且市值在三五億以下，若遇到盤中下殺，必須比別人跑得更快。因為籌碼量大，若不在流動性足夠時先砍，後面會被卡死（出不掉）。[逐字稿 00:42:00]
*   **關於兩性溝通：** 面對不願嘗試「坐著尿尿」的男友，主委建議溝通時要「降規」，讓對方覺得你跟他是在同一隊的。越是嗆聲或比較（如拿主委來壓他），對方越容易產生反感心態而抗拒。[逐字稿 00:49:00]

## 行動建議與觀察清單

1.  **關注 CPU 供應鏈：** 特別是與 AWS Graviton 或 RISC-V 相關的廠商。
2.  **被動元件法說會：** 觀察國巨週三法說會內容，驗證漲價與缺貨趨勢是否擴散。
3.  **調節美股 Exposure（風險暴露）：** 持續加碼強勢族群（CPU、光通、高速傳輸），對走勢破位的長線標的（如 Tesla）進行適度調解，但不需全數清空。
4.  **注意 SMT 代工新勢力：** 留意台股中原本非光通訊本業，但近期獲得美系巨頭委託代工生產光組件的隱藏標的。
